# 说明

## 功能

- 图片分析

## 技术点

- tensorflow

## 项目结构

- app 开发文件
- train 学习的开发文件
- output 学习之后生产的文件
- image 学习的图片
- test 测试用的图片

## 运行

- start: 启动 html 服务
- output：启动生产的训练文件可跨域访问
- run:单纯的训练模型

### 启动顺序

run => output => start

## 步骤

- 样本准备
- 模型构建
- 模型训练

  1.分批读取大数量样本 2.生产模型文件

- 模型预测

## 问题

样本数量有限，会出现不准确的现象，需要将图片作为样本放进图片库里，继续学习

## 拓展

- 目标检测：人脸识别、二维码识别
- 文本生产：在线写小说
